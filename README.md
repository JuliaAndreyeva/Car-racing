## Car Racing Game

This project was developed by students as part of the Continuous Integration and Deployment course.

### Overview
The project is a car racing game implemented using Python's Pygame library. It serves as a practical application of concepts learned throughout the course, including continuous integration, automated testing, and deployment pipelines.

### How to Play

1. Make sure you have Python and Pygame installed on your system.
2. Clone this repository to your local machine.
3. Run the game by executing the main.py file.
4. Use the following controls to play:
   - Accelerate: Press the W key.
   - Decelerate/Reverse: Press the S key.
   - Steer Left: Press the A key.
   - Steer Right: Press the D key.
5. Avoid collisions with the track borders and opponent cars.
6. Reach the finish line before your opponent to win the game.

### Settings

You can customize your gaming experience by choosing different settings using command-line arguments:

- Map: Choose between map1 and map2.

Example usage:
python main.py -m map2 

### Requirements

- Python 3.x
- Pygame
